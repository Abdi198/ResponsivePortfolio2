Travel Japan
project 1

Travel Japan website gives users a look into traveling to Japan during the current Covid-19 pandemic. This was our first project during the bootcamp and worked on it with four other classmates. We divided into front-end team which I was part of and the back-end team. I implented the carousel and parallax using materialize css.

Farmanger
project 2

Farmanger is a database app that allows users to keep track of crops and the land it is planted on. My cohort Patrick and I worked on this app together as a team. Took us about a week to complete it from start to finish.Some of the languages used in building this app include MongoDb, Atlas, Express, Node.JS & Git.

Note Taker
Project 3

Note Taker is an application that allows you to save notes with a title in plain text. Developed this app by myself using ExpressJS. Note Taker app makes creating, viewing, and deleting notes a cinch!
